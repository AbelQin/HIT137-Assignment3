## OOP Image Editor (Tkinter + OpenCV)

## A desktop application developed to demonstrate the use of Object-Oriented Programming (OOP) principles, Tkinter-based GUI development, and image processing with OpenCV.

## Features

Grayscale Conversion

Gaussian Blur (adjustable intensity)

Canny Edge Detection

Brightness Adjustment

Contrast Adjustment

Image Rotation (90° / 180° / 270°)

Image Flip (horizontal / vertical)

Image Resize / Scaling

Save Processed Output

Reset to Original Image
